Keyboard Shortcuts



These work with most text editors as well

  • Cut: Ctrl + X, Copy: Ctrl + C
  • Paste: Ctrl + V, Select All: Ctrl + A
  • Ctrl + ⬅, Ctrl + ➡: Move left or right a word at a time
  • Shift + ⬅, Shift + ➡: Select left or right (works up or down too)
  • Ctrl + Shift + ⬅, Ctrl + Shift + ➡: Select left or right a word at a time
  • Ctrl + Backspace, Ctrl + Delete: Delete previous or next word
  • Home: Start of line, End: End of line
  • PgUp: Top of Document, PgDn: Bottom of Document
  • Double-click text to select that word, triple-click text to select that line
