Shortcuts



These work with most text editors as well

  • Cut: Ctrl + X
  • Copy: Ctrl + C
  • Paste: Ctrl + V
  • Select All: Ctrl + A
  • Move left / right one word at a time: Ctrl + ⬅ / Ctrl + ➡
  • Select left / right character (works up/down too): Shift + ⬅ / Shift + ➡
  • Select left / right word at a time: Ctrl + Shift + ⬅ / Ctrl + Shift + ➡
  • Delete previous / next word: Ctrl + Backspace / Ctrl + Delete
  • Start / end of line: Home / End
  • Page up / down: PgUp / PgDn
  • Double-click text to select that word, triple-click text to select that line
